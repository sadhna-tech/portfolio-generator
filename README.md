# Portfolio Generator

A Python-based tool that generates a responsive personal portfolio website automatically.

## Features

- Generate portfolio website in seconds
- Add personal information
- Display skills
- Showcase projects
- Include contact details
- Creates ready-to-use HTML file

## Technologies Used

- Python 3
- HTML
- CSS

## Project Structure

portfolio-generator/
│
├── portfolio_generator.py
├── portfolio.html
├── README.md

## How to Run

```bash
python portfolio_generator.py
```

## Example Input

Name: John Doe

Role: Python Developer

Skills:
Python, HTML, CSS, Git

Projects:
Weather Dashboard, Resume Analyzer

Email:
john@example.com

## Output

Generates:

```text
portfolio.html
```

Open the file in any browser.
